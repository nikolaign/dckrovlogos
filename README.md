# dckrovlogos

This repository contains the logo(s) and other assets used for the **RADIO KROV public player** and related **Community Center KROV** projects.

## About

Community Center KROV (DC KROV) is based in Belgrade, Serbia. This repository hosts assets for official use on the public radio player and other authorized projects.
